What is Task Tracker?

Task Tracker is a windows 10 program that adds four buttons to the windows right click manager relating to task tracking.

Just a warning, this will not update the file's icon instantly and could take a few minutes to finalize.

Buttons

Default

Finished Task

Hidden task

Unfinished Task

Download

Extract the ZIP, keep the two items in the same folder and run the installer. The program can be uninstalled at any time by re-running the installer.
